#Caesar Cipher / Encryption Tool

A simple Python-based application to encrypt and decrypt messages using the classic Caesar Cipher algorithm. This project demonstrates basic cryptography concepts, string manipulation, and an optional graphical user interface.


##Features
- Encrypt messages using a shift-based cipher  
- Decrypt messages back to original text  
- User-defined shift/key support  
- Simple and easy-to-use interface  
- Optional GUI using Tkinter  


## Technologies Used
- Python (Core Programming)  
- Tkinter (for GUI)  
- Built-in string module  


## How It Works
The Caesar Cipher shifts each letter in the message by a fixed number of positions in the alphabet.

**Example:**
- Plain Text: `HELLO`  
- Shift: `3`  
- Encrypted Text: `KHOOR`  


##  Usage

### 1. Clone the repository
```bash
git clone https://github.com/your-username/caesar-cipher-tool.git
cd caesar-cipher-tool
```

### 2. Run the program
```bash
python main.py
```

### 3. Steps to Use
- Enter your message  
- Choose encrypt or decrypt  
- Provide a shift value  
- View the output
